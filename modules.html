Degree Overview

The degree I am taking is called Artificial Intelligence and Computer Science with a Year in Industry BSc. There are 7 modules throughout the first year and for each of them I provided a short description and a suggestive image below. You can find the official descriptions of the modules here.

Modules

Introduction to Software Engineering (COM1001)

This module introduces basic Software Engineering concepts needed in order to develop simple software systems that have basic functional requirements. The main and most interesting part of this module is the team project in which the students need to develop an app for a "client" who is an older student. The programming language introduced in this module is Ruby.

Foundations of Computer Science (COM1002)

This course develops mathematical concepts and techniques which represent the foundations of computing and are very important for the future computing courses. The propositional logic at the beginning can be rather confusing at a first glance but it gets better once you get a grip on it.

Java Programming (COM1003)

This module introduces programming concepts through Java programming language, as its name suggests. There are a lot of practical exercises in which selection, repetition, object-oriented programming concepts are introduced in order to solve problems. The emphasis is placed on writing a well-structured, efficient and readable code.

Machines and Intelligence (COM1005)

This module provides an introduction to Artificial Intelligence by discussing key concepts, such as whether a computer is capable of understanding and performing human-like tasks based on reasoning. There is a brief historical overview of the subject and reviews the state-of-the-art and open questions in some of the major subareas of AI. There is a hands-on component that involves programming a robot (MiRo) which is exciting but can become very tricky because it involves developing an appropriate algorithm or technique in order to make it perform certain tasks.

Web and Internet Technology (COM1008)

This module covers the basics of networks and internetworking, the importance of the Web in today's society, information about security and technologies used in web development, as well as the best practices when building a website. The most important and interesting part of the module is building a simple and interractive website using HTML, CSS and Javascript.

Introduction to Algorithms and Data Structures (COM1009)

This module introduces students to the design and analysis of efficient algorithms and data structures, alghoritms and problem solving being at the heart of computer science. Students are taught how to analyse the efficiency of an algorithm and how to build efficient ones using different data structures and Programming techniques.

Cognitive Psychology I (PSY1002)

This is an optional module available only for the students taking the Artificial Intelligence course. It covers perception, attention, performance, cognitive neuroscience, language, learning, memory and reasoning, as well as key concepts, theoretical perspectives and foundational methods.
